Mr. Sykes how do you keep focused on your work?

  • Focus Keeper - it is a timer that counts down from anywhere between onw minute to sixty four minutes down.
  • Microsoft To Do - an app that allows you to write down what you need to do for the day, for the week, set dates for when they are due and mark them as important.
  • Use Microsofts Planner is a website that allows you to also write down what you need to do, set due dates and mark them as urgent or however important you think you are.
  • To Test myself I use quizlet, which is a website to setup questions and answers to test yourself. there is an app you can also use, use the same account you used to see all content on all devices.
  • As for note taking applicatinos, I use:
  • Microsoft's One Note,
  • Google Docs (with the speech to text tool).
  • In Google Chrome I use flowtime v4.
  • As a replacement I use Focus Keeper on my iPad or iPhone to help me focus.
